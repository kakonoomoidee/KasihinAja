# v1.2.0 - Angry Toaster

> Released: 2026-03-25

Infrastructure goes interstellar. Contract addresses auto-sync, OBS overlays get a full glow-up, and donations finally arrive in one piece.

## What's New

### Unified Media + Donation Alerts
- Donations with YouTube media now arrive as **one combined card** in OBS (video on top, message bubble attached below)
- Created `pendingMedia.js` in-memory store with 60s auto-expire
- Frontend pre-registers media BEFORE executing the on-chain donation
- `evmListener.js` merges pending media into the `NEW_DONATION` WebSocket payload
- Removed separate `MEDIA_SHARE` WebSocket event entirely

### Auto-Sync Contract Addresses
- Created `Frontend/.env` with `VITE_` prefixed environment variables
- `config.js` now reads from `import.meta.env` instead of hardcoded strings
- Truffle migration auto-writes deployed contract address to both `Backend/.env` and `Frontend/.env`

### OBS Overlay Redesign
- **Alert Overlay**: Glassmorphic combined card with `backdrop-blur` frosted glass, avatar + donor info + message in one unified block
- **Milestone Overlay**: Translucent frosted glass bar (`rgba(255,255,255,0.08)` + `backdrop-blur: 24px`), blue gradient fill with green shift on goal completion
- Both overlays force `document.body.style.background = "transparent"` for clean OBS rendering
- Alerts display one-at-a-time with a sequential queue system (500ms gap between alerts)

### Milestone Auto-Reset
- When `milestone_current >= milestone_target`, milestone resets to 0 automatically
- Streamer starts a new donation cycle without manual intervention

### EVM Listener Fix
- Auto-detects HTTP vs WS URLs (uses `JsonRpcProvider` for Ganache, `WebSocketProvider` for WS)
- Previously silently crashed when using HTTP URLs with `WebSocketProvider`

### Input Validation Fixes
- Donation amount input now has `min="0.0001"` to block negative values
- YouTube "Start at (sec)" input allows clearing the zero before typing a new value

### Changelogs
- Added `changelogs/` directory with versioned release notes and codename index
