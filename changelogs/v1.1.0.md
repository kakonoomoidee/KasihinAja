# v1.1.0 - Nebula

> Released: 2026-03-24

The first major update. Streamer features go live, the UI gets its first glow-up.

## What's New

### Glass-Bento-Gradient UI Redesign
- Full visual overhaul with aurora mesh gradient background (deep navy with teal glow)
- Glassmorphic components (`backdrop-blur-xl`, frosted borders, translucent cards)
- Bento grid layout across dashboard with consistent spacing
- Inter font family via Google Fonts

### Split OBS Overlays
- **Alert Overlay** (`/overlay/alert/:key`) -- Supports Classic and Minimalist templates with customizable colors
- **Milestone Overlay** (`/overlay/milestone/:key`) -- Live-updating frosted progress bar
- Legacy `/overlay/:address` route preserved as fallback

### Media Share System
- Donors can attach YouTube URLs with custom start time when enabled by streamer
- New `/api/media-attach` endpoint relays media data to OBS via WebSocket
- YouTube iframe auto-plays on the Alert Overlay with 30-second timeout

### Milestone Tracking
- Donations now automatically increment `milestone_current` on the streamer profile
- MilestoneOverlay re-fetches on every new donation event

### Session Persistence
- Dashboard login persists across page refreshes via `localStorage`
- Auto-restores wallet session without re-signing on reload
- Session cleared on disconnect or account change

### Backend Enhancements
- Added `enable_media_share`, `enable_vn`, `alert_template`, `banned_keys` columns to `streamer_profiles`
- AI filter now uses streamer-specific `custom_blacklist` from database
- Profile controller extended with `mediaAttach` endpoint

### Bug Fixes
- Fixed double notification on alert overlay
- Fixed Media Share checkbox not persisting after save
- Fixed milestone not incrementing after donation
- Resolved ESLint warnings (`no-unused-vars`, `set-state-in-effect`)
